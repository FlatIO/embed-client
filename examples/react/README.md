# Flat Embed React Example

This example demonstrates how to use `@flat/embed-react` in a React application.

## Running the Example

```bash
# Install dependencies (from repository root)
pnpm install

# Run the example
cd examples/react
pnpm dev
```

## Examples Included

1. **Basic Component Usage** - Using the `<FlatEmbed>` component with event handlers
2. **useFlatEmbed Hook** - Using the `useFlatEmbed` hook for custom UI and more control
