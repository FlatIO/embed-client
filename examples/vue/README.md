# Flat Embed Vue Example

This example demonstrates how to use `@flat/embed-vue` in a Vue 3 application.

## Running the Example

```bash
# Install dependencies (from repository root)
pnpm install

# Run the example
cd examples/vue
pnpm dev
```

## Examples Included

1. **Basic Component Usage** - Using the `<FlatEmbed>` component with event handlers
2. **useFlatEmbed Composable** - Using the `useFlatEmbed` composable for custom UI and more control
